## Descripción
Este trabajo práctico integrador reúne una serie de programas desarrollados en PSeInt y Python con el objetivo de aplicar los conceptos fundamentales de programación aprendidos durante el curso. A lo largo de los distintos ejercicios se utilizaron variables, estructuras condicionales, ciclos repetitivos, funciones, subprocesos, manejo de cadenas de texto, matrices y generación de números aleatorios.

Entre las aplicaciones desarrolladas se encuentran calculadoras matemáticas, procesamiento de texto, manejo de matrices, generación de tablas, conteo de vocales, juegos interactivos y representación de redes de comunicación. Cada programa fue diseñado de forma modular, utilizando funciones y subprocesos para mejorar la organización, legibilidad y reutilización del código.

Como complemento, se realizó una implementación en Python utilizando las bibliotecas NetworkX y Matplotlib para representar gráficamente una red dirigida de ocho nodos, aplicando conceptos básicos de redes y visualización de datos.

Este proyecto permitió integrar conocimientos teóricos y prácticos, fortaleciendo las habilidades de análisis, resolución de problemas y desarrollo de algoritmos.
